I'm pursuing an **Integrated Dual Degree (B.Tech + M.Tech)** in Electrical Engineering at **IIT (BHU) Varanasi**, with most of my work sitting somewhere between computer vision, geospatial data, and applied ML.

- 🛰️ Spend a lot of my time on **vision-language models**, **satellite imagery analysis**, and squeezing ML pipelines onto real-world, messy data

## 🚀 Featured Projects

| 🛰️ Project | 💡 What it does |
|---|---|
| [**GeoNLI**](#) | Vision-language framework unifying visual grounding, captioning, and VQA for satellite imagery — Qwen3-VL-8B + SAM-3 pipeline, GSPO fine-tuning, KAN autoencoder for SAR/IR; built for air-gapped deployment. **Bronze medal, Inter IIT Tech Meet 14.0** |
| [**PCB Defect Detection**](#) | Orthogonal deep neural network (Faster R-CNN + VGG16) for PCB defect detection — **87.2% accuracy** across six defect types, with a novel saliency-based orthogonal regularizer |
| [**Vyapaar AI**](#) | RAG-based assistant for Indian MSMEs covering GST, government schemes, and loan eligibility — combines FAISS retrieval with live web search and Gemini for up-to-date, low-hallucination answers |
| [**Groundwater Reconstruction (IIT Gn)**](#) | ML framework (Extra Trees + K-means) reconstructing 20 years of monthly groundwater depth data across 4,300+ wells in India, **R² = 0.82** |
